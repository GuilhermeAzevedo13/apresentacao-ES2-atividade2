Sobre o Modelo

O mDeBERTa-v3-base-xnli é um modelo de classificação zero-shot que analisou resumos de hipóteses e definições teóricas do projeto AnythingLLM.

Diferente dos outros modelos, ele não requer treinamento específico para cada categoria, permitindo classificação direta sem exemplos rotulados.

Aplicações no Projeto

• Análise de estratégias de branching e release

• Classificação semântica de conceitos teóricos

• Identificação de padrões de desenvolvimento

Características Principais

Classificação Semântica

Capacidade de entender relações de hipótese e definição

Zero-Shot Learning

Classificação direta sem necessidade de exemplos rotulados

Probabilidade Semântica

Retorna valores de probabilidade para cada classificação
