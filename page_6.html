Sobre o Modelo

Modelo de classificação por texto para analisar padrões em arquivos do repositório AnythingLLM.

Dados Analisados

  • Arquivos brutos do repositório
  • branches_overview.txt
  • Logs de commits

Como Funciona

Utiliza logits negativos para classificação (quanto mais próximo de 0, melhor).

Compara padrões textuais com modelos de branching e release.
